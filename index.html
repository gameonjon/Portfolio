Roll A Ball:

Here we have a simple tutorial execution of Roll A Ball. The engine used to create this was Unity. Tutorial can be found at here. This game includes:

  • Pick-up Objects
  • Score Board on top left
  • End game pop-up
  • Basic play arena



Videogame Database:

This project was created to display a small number of available videogames. I used Python as the backend to create the schema; MySQL was used as the RDBMS. Nodejs/Expressjs was used as the Rest API to pass requests I then used HTML and CSS to create the frontend to display the information currently in store. This projct can be found in my Github repos.



Gami Bird:

Here is another Unity tutorial execution. This is based off of the popular game, Flappy Bird. Tutorial can be found here. Besides just getting a better understanding for 2D game development, the main purpose was to practice publishing/shipping of game products. This game can be found on Google Play here.
